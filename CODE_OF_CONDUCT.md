# Code of Conduct

This project adopts the **Contributor Covenant** (v2.1) as its code of conduct. The full text is
published at:

- https://www.contributor-covenant.org/version/2/1/code_of_conduct/

In short: be respectful and constructive; harassment and discrimination are not tolerated.

## Enforcement

To report unacceptable behavior, contact the maintainer privately at **me@rubentalstra.nl**. All
reports are reviewed and handled confidentially.
